## Features

- **Hero Section**: Interactive introduction with animated name, professional titles, and value proposition.  
- **Experience Timeline**: Visual representation of internships, projects, and professional training.  
- **Skills Section**: Technical skills with progress bars, categorized for clarity.  
- **Education Section**: Academic and online learning background, including certifications.  
- **Certificates Section**: Showcase of achievements and verified certifications.  
- **Projects Section**: Highlight of personal and academic projects with links.  
- **Contact Section**: Reach me via email, social links, or a contact form.  
- **Responsive Design**: Mobile-first, fully responsive layout.  
- **Modern UI**: Tailwind CSS with glassmorphism and animated interactions.

## Technologies

- **Frontend**: React, Next.js  
- **Styling**: Tailwind CSS  
- **Icons & Animations**: Lucide React, CSS animations

- **Deployment**: Vercel / Netlify (optional)
